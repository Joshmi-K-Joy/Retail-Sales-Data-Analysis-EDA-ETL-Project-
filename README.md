
# End-to-End Retail Data Analytics Project

## Business Objective
Analyze retail sales data to identify performance drivers, loss-making areas, customer behavior, and forecast future sales to support business decisions.

## Tools & Skills
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Excel
- Power BI
- Statistics & Forecasting
- GitHub Project Structure

## Project Steps
1. Business Understanding
2. Data Collection
3. Data Understanding
4. Data Cleaning & Preparation
5. Exploratory Data Analysis (EDA)
6. SQL Analysis
7. Dashboarding (Power BI)
8. Forecasting
9. Insights & Recommendations

## Folder Structure
Refer to repository folders for notebooks, SQL scripts, dashboards, and reports.

## Key Insights (Sample)
- Identified 12,543 loss-making transactions driven by discounts and shipping costs.
- Top 20% products contributed ~65% of total profit.
- Forecasted next-quarter sales trend using regression.

## How to Run
1. Place raw dataset in data/raw/
2. Run cleaning notebook/script
3. Use cleaned data for EDA, SQL, and dashboards
